# 關於講師

# AI 趨勢 (1)：從 Chatbot 到 Agent
2024 年：你問，它答（像是圖書館管理員）。
2026 年：你下指令，它去執行任務（像是你的專業秘書）。
AI Agent (智能體) 具備自主拆解目標、尋找工具、完成回報的能力。

# AI 趨勢 (2)：協作模式的改變
人不再是「操作者」，而是「驗收者」。
你的工作從「親手做」變成「給評論」。
AI 做 80% 的粗活，你負責最後 20% 的精修與決策。

# OpenClaw AI Agent？
概念比喻：一個擁有「爪子 (Claw)」的大腦。
它能透過 API 或模擬操作，去觸碰外界的工具（試算表、網頁、郵件）。
核心： 模組化、可替換、易於控制。

# 未來工作只有兩種
Human in Loop (人機協作)
Human out of Loop (AI自主)

# 未來的教室會變怎樣？
每個學生都有一個根據其程度客製化的「 24 小時助教 」。
老師的角色：這些助教的「總設計師」。

# Prompt (提示詞) vs Prompt Engineering (上下文工程)
不是咒語，是「工作說明書」。
你對助理怎麼說話，就對 AI 怎麼說話。

# 技術平權時代的到來
以前：想做個網頁，得學三年程式碼。
現在：只要能把「邏輯」說清楚，AI 幫你寫網頁。
結論： 技術門檻正在消失。

# 當技術不再是門檻，什麼最重要？
答案：好的想法 (Good Ideas)。
懂學科專業知識的你，比懂程式碼的工程師，在 AI 時代更有優勢。

# 你想解決什麼問題？ (痛點發掘)
範例：改不完的通識報告？學生聽不懂的專有名詞？沒人看的教學網站？

# 認識 Google AI Studio：開發者的實驗室
為什麼選它？
因為它最直覺、性能最強大（Gemini多模態模型）、而且目前對一般使用者最友善。
